# Cockroach and lamp
Requirements
--------
    python 2.7
    pygame 1.9.1

How to run
----------
    python main.py
    
Controls
--------

**Cockroach** 

    A: Go left
    
    D: Go right

**Lamp**

    Left Arrow: Move left
    
    Right Arrow: Move right
    
    Spacebar: Turn on/off
